# trello-clone

Steps to run:

1. Clone the repo.
2. Go to /trello.
3. Run npm install && bower install.
4. Run gulp serve

Technologies used:

1. Yeoman -> For folder structure generation.
2. Gulp -> To minify and uglify also to append .js and .css files.
4. Bower -> Dependency build maintenance.
3. MVC -> AngularJs
